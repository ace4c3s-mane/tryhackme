
Some of the commands or methods that my web browser commonly issues to the web server are:

1. GET retrieves data from a server, such as an HTML file or an image.

2. POST allows us to submit new data to the server, such as submitting a form or uploading a file.

3. PUT is used to create new resource on the server and to update and overwrite existing information.

4. DELETE is used to delete specified file or resource on the server.


HTTP and HTTPS commonly use TCP ports 80 and 443, respectively, and less commonly other ports such as 8080 and 8443.
