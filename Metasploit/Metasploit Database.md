
metasploit has a database function to simplify project management and avoid possible confusion when setting up parameter values.

metasploit uses the PosgreSQL database. When active, it allows you to create workspaces and isolate different projects.

When first launched, you should be in the default workspace. you can list available workspaces using the workspace command.

```bash
workspace
```
